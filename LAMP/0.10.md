#route命令
####route
route命令是管理路由表的            
```text
route -f
```
清除所有网关入口的路由表，以至你的网络马上出现故障，不能上网
```text
netsh int ip reset
```
执行以上命令后，即重置TCP/IP协议后，重启电脑，即可恢复正常
```text
route -p ADD
```
该命令可以使添加的路由表为永久路由表信息，重启电脑后依然存在，而不加`-p`的路由信息，重启电脑后就不存在了

添加路由
```text
route ADD destination MASK mask gateway METRIC metric IF Interface
route ADD 0.0.0.0 MASK 0.0.0.0 192.168.0.1 METRCI 276 IF 3
route ADD 0.0.0.0 MASK 0.0.0.0 192.168.0.1 IF 3
route ADD 0.0.0.0 MASK 0.0.0.0 192.168.0.1
```
destination是目的IP地址        
mask是子网掩码        
gateway是网关         
metric是跳数         
Interface是网络接口，在`route print`打印出来的接口列表中，最前面的那个数字            

删除路由
```text
route DELETE 0.0.0.0
route DELETE 0.0.0.0 MASK 0.0.0.0
route DELETE 0.0.0.0 MASK 0.0.0.0 192.168.0.1
```

修改路由，只能修改网关和越点数
```text
route CHANGE 0.0.0.0 MASK 0.0.0.0 192.168.0.11
route CHANGE 0.0.0.0 MASK 0.0.0.0 192.168.0.11 METRIC 2
```
