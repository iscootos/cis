###变量
使用`var`关键字声明变量,没有设置值；默认为`undefined`，用`var`声明的变量都是局部变量，推荐所有变量都使用`var`声明
```js
var i;
```
使用`var`关键字声明并赋值，值可以是整形，布尔值，字符串，数组，对象，函数
```js
var i = 10;
var i = "hello world";
```
隐式声明，全局变量
```js
i = 'hello wolrd';
```
用`window`关键字声明的全局变量
```js
window.i = 'a';
```

一次性声明多个局部变量
```js
var a=1,
	b=2,
	c=3;
```
