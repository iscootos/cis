#字符串
####创建字符串
字符串使用`'`单引号或者`"`双引号括起来
```js
var x = 'hello world';
var y = "hello world";
```
####连接字符串
使用`+`运算符，可以连接字符串
```js
var a = "hello ";
var b = "world!";
alert(a+b);
```
#####indexOf()
返回某个指定的字符串值在字符串中首次出现的位置
```js
var x = 'abc';
var y = x.indexOf('b');
var z = x.indexOf('h');
alert(y);	//输出 1  	从0开始，所以b是1
alert(z);	//输出 -1	没有找到该字符串，则返回 -1
```
