#elment对象
####appendChild()
向元素添加最后一个子节点
####attributes
返回指定节点的属性集合
```js
var x = document.getElementById("myBtn").attributes.length;
```
####childNodes
返回节点的子节点集合，NodeList对象
```js
document.body.childNodes.length;
```
####className
设置或返回元素的class属性
```js
document.write(document.getElementById('myid').className);
```
