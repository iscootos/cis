#创建数据表
创建数据表的语法形式，`CREATE TABLE`不区分大小写
```text
CREATE TABLE 表名称
(
	列名称1  数据类型  列约束条件  默认值,
	列名称2  数据类型  列约束条件  默认值,
	....
	表约束条件
);
```
例如我们要创建一个表，表结构如下，表名称person
```text
字段名称        数据类型           备注
id              INT(11)            编号
user            VARCHAR(25)        用户名
passwd          VARCHAR(25)        密码
```
那么它的SQL语句如下
```sql
CREATE TABLE person
(
	id INT(11),
	user VARCHAR(25),
	passwd VARCHAR(25)
);
```
创建完成后，使用如下命令，查看数据表
```sql
SHOW TABLES;
```
