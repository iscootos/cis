<h1>Sublime Text 3常用命令</h1>

`Ctrl+L`           选择整行（按住-继续选择下行）                           
`Ctrl+KK`          从光标处删除至行尾                                      
`Ctrl+Shift+K`     删除整行     
`Ctrl+Shift+D`     复制光标所在整行，插入在该行之前     
`Ctrl+/`           注释整行（如已选择内容，同“Ctrl+Shift+/”效果）        
`Ctrl+Shift+/`     注释已选择内容    
`Tab`              缩进    
    
`Insert`           替换当前字符    

#####注释
`Ctrl+/`			注释整行			
`Ctrl+Shift+/`		注释代码块			

#####折叠、展开代码
`Ctrl+Shift+[`		折叠代码			
`Ctrl+Shift+]`		展开代码
