###JavaScript变量
变量名:大小写敏感   
var 变量名 = 值

    var s;                    //没用设置值；默认为undefined
    var a = '';               //空字符串
    var b = 12;               //整型
    var c = 'hello world!';   //字符串
    d = 'hello';              //直接赋值，而不声明变量。（不推荐）
    window.var;                     //声明全局变量
    window.var = 'hello world';     //声明全局变量并赋值

创建多个变量：   
实例1：

    var a = 1,
        b = 2,
        c = 3;

实例2：

    var a = 1,b = 2,c = 3;

JavaScript 布尔    
布尔（逻辑）只能有两个值：true 或 false。  

    var x = true;
    var y = false;
